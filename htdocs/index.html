About

What is GraphTV?

GraphTV is a tool for visualizing IMDb series ratings. A lot can be told about a series from looking at episode ratings and season trends.

You can quickly answer some interesting questions like:

How has a long running series aged? The Simpsons
Did it take a season or two before the writers hit their stride? Parks and Recreation
What's the best episode in an anthology series? Black Mirror
Was one episode really out of place in a series? Stranger Things
How are the series' season finales? Orange Is the New Black

It is HEAVILY inspired by Kevin Wu's pioneering website GraphTV which doesn't appear to be under development any longer.

Tips?

Colors indicate season
Trend line shows the season's rating over time
Episode dot sizes based off the relative number of votes for that episode
Tooltips show Season #, Episode #, Title, Rating, and Votes
Click and drag to zoom in on busy areas

Development

The code for the website and all back-end functionality is FULLY open sourced under the GPLv3 License. The source code is hosted in the GraphTV GitHub Organization maintained by Caesar Kabalan. The goal was to build a scalable web applications using current cloud architectures. As many best practices as possible were followed and are all documented in the GitHub Org.

Any and all contributions are valued! The best way to contribute is in the form of a GitHub pull request or issue. No contribution to small, even fixing typos in documentation!

A brief overview of technologies used to bring you this web application:

Amazon Web Services for infrastructure hosting:
CloudFront for content delivery
API Gateway as an API Endpoint
Lambda for serverless back-end
DynamoDB as a show/episode database
Route53 for dynamic DNS
Certificate Manager for free SSL certificates
S3 for content hosting and storage
CloudFormation for deploying every service above
Programming Technologies:
Python 3.6 for back-end services
Serverless Framework for AWS Lambda deployments
Semantic UI as the CSS Framework
Simple Statistics for the Linear Regression calculations
Highcharts for the main chart
jQuery for general Javascript usage
Data Sources:
IMDb Interfaces for all series, episode, rating, and vote data
